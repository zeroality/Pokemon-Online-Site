PO Wiki - All systems are go

Written by eric the espeon on 24/Sep/2011, 11:46 AM

The Pokemon Online Wiki is live and public.

It is of course very, very far from complete, though you may already find a good amount of useful content (over 160 analyses, full all-gen movepools for every Pokemon along with other competitive data on the species pages, type chart, and plenty of other fun pages). What has been written so far is merely a skeleton of what is to come, and we need your help to make that a reality. If you share our aims; to create a great resource for competitive Pokemon players, one which is open, embraces radical trust, is inviting, works with fellow communities, and is inclusive of a wide range of relevant information, or just want to build a part of something awesome, then you are invited to join. Find us on #PO Wiki on the main PO server, or on #PokemonOnline on irc.synirc.net.

Any user is welcome to edit any page they feel they can improve, or to create pages which would be useful to other players. Every page has a full edit history saved, so nothing can be permanently broken or lost, and any damage from harmful edits can be easily repaired. As the community develops and content matures, staff will be appointed who will watch over and ensure the quality of various pages and sections.

I'd like to thank a few of many contributors who made this possible, Jirachier for setting the wiki up initially and implementing the skin, Darkness for creating the logo and skin/main page design, coyotte508 for creating the program which this community is built around and asking me to build the wiki as well as his work on some important templates and data imports, zeroality for making things more efficient, and nixeagle for his work on importing base pages for items/moves/species/abilities as well as many other things, but perhaps most importantly convincing me to stick with it and make this thing ready to go public. I'd also like to thank all those writers, editors, and templaters who've put together most of what you will find, and provided many ideas to improve every aspect of the wiki. Keep it up people

The site is still being improved, nixeagle and zeroality are working hard behind the scenes to optimize and add features which you should see popping up soon, but everything we need to build content is in place. So, come on over and play, let's fix everything!

So instead of introducing new version numbers for everything, tags!

The first patch of 1.0.30 introduces some improvements:

Client Side:
- Channels don't 'flash' on events anymore (battle start, etc.)
- Clients are able to display pokemon sprites in announcements and chat
- More customizable stuff with themes.
- Reverse text characters blocked from battles & chat

Server Side:
- Server bug about orbs is fixed
- Technician now works on struggle
- Server bug about tailwind is fixed

How to display pokemon sprites?
<img src="pokemon:num=151&shiny=false&gender=neutral&ba ck=false&gen=2" />
<img src="item:20" />

For pokemon, all the parameters except the pokemon num are optional. Formes can be accessed with 493-4, for example.

So, if you want to display a mew in your trainer info, just do:
<img src="pokemon:151&gen=2"/> for an old school%2

1.0.30

Written by coyotte508 on 29/Aug/2011, 02:48 AM

Version 1.0.30 comes, after things like android client and replay viewer!

Reminding of android client: It's accessible here and it's thread on PO here. It allows you to behave like a PO client and all, meaning chat, channels, and battles! It can't teambuild yet, but since last PO update it can load teams on PO through the use of QR Codes. <3 Android developers.

About the replay viewer, it was made by aeo. It allows you to convert PO log files into animated battles on your browser which look awesome. Since the latest update, changes to the log files have been made to facilitate the work of the replay viewer. It has been changed a bit though, so aeo may need a bit of time to update the replay viewer, but then it'll work great!

So that's it. Other changes have been made since last update, like the teambuilder loading fast now, and numerous bugfixes.

Next version will possibly see major overhauls, making it incompatible with all previous versions but introducing new things such as announcement on registry (to notify of new versions), possibility to reconnect on a server after being disconnected, including during a battle, without loss of data, for people with shaky connections; the RBY version implemented by nixeagle, possibilities to load several teams at once, and maybe custom pokemons made a lot easier per server and loading automatically! We'll see, but there's definitely major room for improvement, and a number of associated projects going on.

Anyway, bugfixes & new features:

Features:
-Team builder loading a lot faster
-QR Code plugin, plugin manager
-Log format containing more info
-Channel red when pinged in a channel (Viderizer)
-Server in System Tray (Latios)
-Teams given in the log window in wifi battles
-Possibility to ignore opponents in battles
-More customizable stuff in Themes

Bugfixes:

- Possibility to select text in chat windows in peace now
- Damage formula use correct random generation mechanism (I'm the culprit behind the misconception for all the DP/BW formula, sorry for that). It had really minor effect though.
- Intimidate no longers intimidates non adjacent foes in triples
- Telekinesis now ends correctly
- Pickup now works correctly
- Sorting by auth fixed (Viderizer)
- Pickup, Recycle & Harvest now work correctly together
- Copycat was fixed after I broke it
- Spiky pichu can't be gotten in CC gen 5 anymore
- Chingling's Egg move combinations were fixed
- sys.rand()'s division per zero crash fixed (Lamperi)
- King rock now effects Icicle spear
- End of turn orders customized per gen and all correct now
- Shedinja no longer invincible when using false swipe
- Protect rate works correctly now in all gens. It always had decreasing rates, it seems, though in gen 4 the decreasing stopped pretty fast.
- Metronome's list of callable moves fixed for gen 5 (seems like it can't call V-Create or the ultimate ice moves)
- Skill Swap was absent from gen 3, added to all the pokemon learning it in gen 3
- Status immunity fixed in gen 3, now ice pokemon can be frozen and fire pokemon burnt by tri attack
- Hex now does deal double damage on pokemon behind substitutes
- Partial trapping moves now don't overwrite each other

- Sap sipper now boosts attack before leech seed fails
- uproar now ends correctly (I pray)

- Healing wish now heals before entry hazards

- Fury cutter adapted depending on gen: in gen 3 & 4, it doesn't reset power when changing moves, while it does in gen 5 (it always resets powers when it fails/misses though)

So, should you get the new version of the client? If you still have the version where the server takes several seconds to load, if you have an android and don't have the version with the QR Code plugin, or if you want to use aeo's awesome replay, then yes!
